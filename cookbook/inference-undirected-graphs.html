Bayesian Inference of Stochastic Block Models#

Forward (generative) process#

In the stochastic block model (SBM) as implemented in graph-tool, the forward generative process begins by assigning each vertex \(v\) to a block \(b_v \in \{1, \dots, B\}\), where \(B\) is the number of blocks.

Given the block assignments \(\{b_v\}\), edges are then placed independently between pairs of vertices \((u, v)\) with a probability that depends only on their blocks, specifically according to the entries of a block-to-block edge count matrix \(e_{rs}\), where \(r = b_u\) and \(s = b_v\).

In the degree-corrected variant, the expected degree of each vertex is further controlled by a parameter \(\theta_v\), and the edge probabilities are proportional to \(\theta_u \theta_v\).

This generative model defines the joint distribution over the graph and block structure, forming the basis for subsequent inference.

Mixing patterns#

B = 6; N = 50
cmap = matplotlib.colors.ListedColormap(np.vstack((colors[:B]))/255.0)
in_group = [0,1,2]
out_group = [3,4,5] # periphery out_group 
mat = np.zeros((B,B))

for i in range(B):
    for j in range(i, B): 
        if i in in_group and j in in_group: 
            mat[i,j] = 0.01
            if i == j: 
                mat[i,j] = 0.02
        if i in out_group and j in out_group: 
            if i == j: 
                mat[i,j] = 0.99
        if i in in_group and j in out_group: 
            if (i == 0 and j == 5) or (i == 1 and j == 3) or (i == 2 and j == 4): 
                mat[i,j] = 0 
                
            else: 
                mat[i,j] = 0.005

mat = mat+mat.T - np.tril(mat) + np.tril(mat, -1).T
def prob(a,b):
    return mat[a,b]
probs = []
for i in range(mat.shape[0]):
    for j in range(i, mat.shape[0]):
        probs.append((i,j, mat[i,j]))
